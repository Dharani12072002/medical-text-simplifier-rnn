# Medical Text Simplification using BART

This project simplifies complex medical text into patient-friendly language using NLP and Transformer models.

## Features
- Medical text simplification
- BART Transformer model
- Medical terminology replacement
- ROUGE evaluation
- Interactive user input

## Technologies Used
- Python
- HuggingFace Transformers
- PyTorch
- Google Colab
- NLP

## Example

Input:
The patient suffered an acute myocardial infarction.

Output:
The patient had a heart attack.

## Installation

pip install -r requirements.txt

## Run

python app.py
